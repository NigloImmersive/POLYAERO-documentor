# Panneau de collaboration

## Local user settings

![LocalUserSettings](/img/manuel-vr-user/manip-pc/collabLocalSetting.png)

- **Alias** : Définissez l'alias sous lequel les autres vous voient dans la simulation.
- **Color** : Définissez la couleur de votre "fantôme" dans la simulation. Par défaut, ce sera également la couleur de votre stylo lorsque vous ferez des annotations en VR.
- **Logo** : Définissez l'image que vous souhaitez affiçcher au dos des mannequins. Votre image doit être carré. Vous pouvez réinitialiser le logo en cliquant sur le bouton de réinitialisation.

## Remote users settings

![RemoteUserSettings](/img/manuel-vr-user/manip-pc/collabRemoteSetting.png)

- **See others as humain** : basculer si les utilisateurs distants sont affichés ou non à l'aide d'un avatar ou d'un mannequin.
- **Reste human** : Recalculer ma position du squelette de chaque mannequin de la scène.

## Users

![Users](/img/manuel-vr-user/manip-pc/collabUser.png)

Vous pouvez voir et administrer les utilisateurs. Les fonctions disponibles sont :

- **Mute** un utilisateur (si VOIP est activé).
- **Fit** votre vue à un utilisateur.
- **Kick** un utilisateur en dehors de la session.

A droite vous pouvez voir :

- **network connection quality** (basé sur la réponse ping)
- **users rights** : **S** = Serveur, **A** = Admin, **U** = User. U et S peuvent déconnecter et couper le son d'autres utilisateurs.

## VOIP

![VOIP](/img/manuel-vr-user/manip-pc/collabVOIP.png)

- **Join/Leave call** : basculer dans la session VOIP.
- Désactiver votre microphone.
- Basculer le son de votre haut-parleur.
- **Input** : sélectionner un périphérique d'entrée spécifique.
- **Output** : sélectionner un périphérique de sortie spécifique.

Si vous ne le voyez pas, allez dans les paramètres et activez la VOIP (serveur et client).
