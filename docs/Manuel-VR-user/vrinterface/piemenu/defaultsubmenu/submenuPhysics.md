# Sous menu de physique

Dans ce sous-menu, les commandes suivantes permettent de travailler avec Physique :

![submenuPhysics](\img\manuel-vr-user\vrinterface\piemenu\physicsDefault.png)

## ![PhysicsIndicator](\img\manuel-vr-user\manip-pc\icons\physicIndicator.png) Physics Indicator

Vous permet de basculer l'état physique d'un objet (Vert = physique activé, Rouge = physique désactivé).

## ![ToggleGravity](\img\manuel-vr-user\manip-pc\icons\toggleGravity.png) Toggle Gravity

Activer/désactiver la gravité. Tous les objets dont l'état physique est activé seront soumis à la gravité.

## ![TogglePhysics](\img\manuel-vr-user\manip-pc\icons\togglePhysic.png) Toggle Physics

Activer/désactiver la physique. C'est le commutateur principal pour utiliser la physique. Une fois activé, tous les objets dont l'état physique est activé seront soumis à la physique (et s'il est activé : la gravité).
